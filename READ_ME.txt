Program tested on Windows 8.1 with Python 3.4

How to run BitCoinArb:

1.open a command windows in folder which contains the files

2.launch the program:
$python bitcoinarb_shell.py
That will launch another shell accepting command

3.Type in "cmdlist" to get list of available commands


Configuration files:

exchanges.xml - Describes all exchanges considered by the program
assets.xml - List all existing assets (crypto and real currencies)
pairs_filter.xml - Used by the program to filter on currencypairs
